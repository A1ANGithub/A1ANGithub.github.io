最长上升子序列

预备知识

  • 动态规划
  • 贪心算法+二分查找

    问题

    给定一个无序的整数数组，找到其中最长上升子序列的长度。

    示例

    Solution Code

    方法一

    解析

    时间复杂度为O(n^2)
    使用动态规划；定义dp[]数组，dp[i]为第i个数字结尾的最长上升子序列的长度；遍历到nums数组第i个数，在用j遍历0–i，若nums[i]>num[j]，把dp[j]+1赋值给dp[i]，直至遍历结束（其中dp[i]取最大值）。最后遍历dp[i]数组，最大值即为最长上升子序列。

方法二

解析

时间复杂度为O(nlogn)
使用贪心算法+二分查找；维护一个数组dp[]，首先讲nums[0]放入，初始长度为1；遍历nums，依次对nums[i]进行处理；若nums[i]大于dp末尾值，讲nums[i] append入dp；若nums[i]小于dp末尾值，则采用二分查找找到第一个比它小的值，并将nums[i]替代该值的下一位置的值（注意若都比nums[i]大，则放在首位）。

总结

发现第二种方法虽然时间复杂度低，但是略微费空间，以下原文链接有一位力扣成员使用C语言编写，空间时间均接近双百
